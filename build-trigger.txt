Extended-storage Saturn SD firmware build

Hardware allocation (PMOD3 fully reserved for SD):
- SD CS:   PMOD3 Pin1 / GPIO6
- SD MOSI: PMOD3 Pin2 / GPIO8
- SD MISO: PMOD3 Pin3 / GPIO47
- SD SCK:  PMOD3 Pin4 / GPIO12

Build configuration:
- SPI3 host
- 4 MHz SDSPI maximum
- POSIX-backed Jaculus fs implementation
- Late persistent /sd mount across Jaculus Machine restarts
- Boot application directly from /sd/robodeck/software/main.js
- Keep Jaculus runtime and installed libraries internal
- Use ESP32-S3 Octal PSRAM through malloc/new for large C++ source buffers
- No JavaScript launcher module and no copy of application code into internal storage

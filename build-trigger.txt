Stable Saturn SD firmware build

Hardware allocation (PMOD3 fully reserved for SD):
- SD CS:   PMOD3 Pin1 / GPIO6
- SD MOSI: PMOD3 Pin2 / GPIO8
- SD MISO: PMOD3 Pin3 / GPIO47
- SD SCK:  PMOD3 Pin4 / GPIO12

Build configuration:
- SPI3 host
- 4 MHz SDSPI maximum
- POSIX-backed Jaculus fs implementation
- Late mount after Jaculus feature initialization
- Preserve successful /sd mount across Jaculus JavaScript Machine restarts
- Do not unmount/remount SD on Build/Flash/Monitor program restart
